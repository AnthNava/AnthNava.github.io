About Me

Anthony Navarro is a recent graduate from Florida International University with a versatile skill set in multiple disciplines. Graduating with a bachelors in information technology, Navarro has a multitude of skills IT such as Full-Stack Development, Programming, and IT troubleshooting. During this time he made projects such as SQL/MongoDB Databases and a Full-Stack Website solution.

Anthony Navarro has a deep passion for game design. Ever since he could remember he has loved video games, and that love grew into a desire to learn and understand game design deeper. In his spare time Navarro pursued this passion though projects such as participating in level design workshops, game consulting, research projects, and deep design discussions with a dedicated community of professional and indie game designers.

These two disciplines inform Navarro's way of thinking and problem solving abilities. He's ready to take on new challenges and wants to be there to help any team succeed in their vision.

My Work

Budget Tracker

A fullstack Python web application that I created. I used flask for the backend RESTful API that would take in user provided data and connect to a MongoDB database to update the data. The program would then calculate the total expenses of all entries and categories and update the front page. The application also allows you to set the type of currency from a provided selection when adding an entry and the Application will convert to USD based on up to date values provided by making calls to the currencylayer API.

Mario Maker Workshop

The Mario Maker Workshop is an online game design school program that used Super Mario Maker 2 as a tool to teach level design through a communal learning method. The Workshop worked in a communal matter that allowed for open feedback on user created levels. Over the course of 16 weeks I worked through this program and developed new understandings of not just the level design of mario but also my own understandings of game design as a whole.
